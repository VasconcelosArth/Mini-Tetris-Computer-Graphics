Este é um projeto para a disciplina de Computação Gráfica da Universidade Católica de Pernambuco

__Objetivo:__ Criar uma __cena ou aplicação 2D__ interativa usando HTML, JavaScript e Canvas 2D que demonstre o domínio das transformações geométricas.

__Requisitos:__

- __Translação__ (ctx.translate) - Mover objetos na cena ✅
- __Rotação__ (ctx.rotate) - Girar pelo menos um objeto ✅
- __Escala__ (ctx.scale) - Redimensionar objetos ✅
- __Composição de Transformações__ - Combinar pelo menos duas operações em sequência (Ex.: Rotação + Translação) ✅
- __Rotação ou Escala com Ponto Fixo__ - Pelo menos um caso o padrão T ⟶ Op ⟶ T ✅
- __Animação__ - Usar requestAnimationFrame com reset de matriz a cada frame ✅
- __Save/Restore ou setTransform__ - Gerenciamento correto da pilha de estados ✅

__✦ Bônus:__

- Interatividade com teclado ou mouse ✅
- Uso de reflexão ou cisalhamento
- Hierarquia de Transformações: Quando um objeto filho herda a transformação do objeto pai (Ex.: Mão acompanha o braço; Lua acompanha o planeta)
